# AI エージェント運用ガイドライン (VeloceDraft)

## 基本原則
1. **停止の禁止**: 日常的なルーチン作業（ファイル作成、スクリプト実行、テスト、ブラウザ確認等）でユーザーに対して確認待ちの停止を行わず、最後まで進行すること。
2. **SSoTの遵守**: すべての設計判断において `AGENTS.md` に記載された哲学・技術スタック・制約を最優先すること。
3. **言語仕様**: 実装およびコード内のコメントは英語。外部向け・計画用のドキュメントは日本語とする。
4. **WASM不使用**: 複雑さを避けるため、フロントエンドでのWASM利用やファイルパースを禁止し、すべてバックエンドの ezdxf へ委譲すること。

## 破壊的操作の確認
以下の操作を実施する場合のみ、処理を停止してユーザーに明示的な許可を求めること。
- 再帰的な削除（rm / del 等）
- git の破壊的操作（reset --hard, clean -fdx, force push）
- シークレット情報の取り扱い
- 外部への送信、またはデプロイ
